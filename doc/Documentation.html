Installation

gcc 2arr.c -o 2arr
sudo mv 2arr /usr/local/bin/
gcc 2arr.c -o 2arr.exe

Usage

2arr [OPTION] ARCHIVE_NAME [FILES...]
Option Description
-p Pack files into archive
-u Unpack archive
-l List archive contents
-h Show help message

Examples

Pack files

2arr -p docs.2arr report.txt images/ src/

Outputs: ./docs.2arr

Unpack archive

2arr -u backup.2arr

Outputs: Unpacked to backup/ successfully

List contents

2arr -l project.2arr

Outputs directory listing

Archive Format Specification

[D|F]path\n
[filesize\n]?  ‚Üê Only for files
[file data]?   ‚Üê Only for files
  • Directories marked with D prefix
  • Files marked with F prefix + size
  • All paths stored relative to archive root
